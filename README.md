## Zustand Setup

We used **Zustand** for simple and efficient global state management in React.

### Installation
```bash
npm install zustand
```

- Installed **Zustand** for global state management
- Created a central **Zustand store**
- Added **state** to hold application data
- Implemented **actions** to update state
- Used **persist middleware** for state persistence
- Stored data in **sessionStorage**
- Accessed store directly in components using a hook
- Eliminated **props drilling** and **Context API**
- Achieved clean, simple, and scalable state management
